# Rock Paper Scissors

A simple command-line Rock Paper Scissors game built using Python.

The player competes against the computer in a best-of-five match. The computer makes random choices, and the program keeps track of wins, losses, and draws before announcing the overall winner.

## Features

- Play 5 rounds against the computer
- Random computer moves
- Keeps track of wins, losses, and draws
- Displays the final game result

## Technologies Used

- Python
- random module

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

```bash
python Rock_Paper_Scissors.py
```
